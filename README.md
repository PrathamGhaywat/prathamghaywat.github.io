# prathamghaywat.github.io

A single‑page portfolio for Pratham Ghaywat — student, AI specialist, and full‑stack developer.

## Preview locally

You can open `index.html` directly in your browser. For a local server with live reload on Windows PowerShell:

```powershell
# Option 1: Python 3
python -m http.server 8000

# Option 2: Node (if installed)
npx serve . -l 8000
```

Then open http://localhost:8000

## Content
- Hero with contact links
- Projects (Sophos, OpenPlant, Promptsmart, PyBrowse, AudioWiz, Devspawn)
- About and Skills
- Education and Certificates
- Languages and Contact cards

## Customize
- Edit `index.html` sections
- Tweak theme tokens in `assets/styles.css`
- JS interactions in `assets/script.js`

## Deploy
This repository is already set up for GitHub Pages. Push to `main` and visit:

- https://prathamghaywat.github.io
